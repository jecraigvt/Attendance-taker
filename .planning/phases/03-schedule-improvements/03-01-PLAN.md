**Part A: Add generate_daily_summary function to sync_utils.py**

Add a new function after generate_verification_report:

```python
def generate_daily_summary(date_str: str, output_dir: str = ".") -> Dict:
    """
    Generate end-of-day summary report aggregating all sync runs

    Args:
        date_str: Date in YYYY-MM-DD format
        output_dir: Directory to write report (default: current)

    Returns:
        Dict with daily totals and breakdown by sync run
    """
```

The function should:
1. Read all audit entries for the date using get_audit_entries(date_str)
2. Read sync_errors_{YYYY-MM}.log and filter to today's entries
3. Read failed_students.json to get any unresolved failures
4. Aggregate totals:
   - total_students_processed (unique student_id + period combinations from intents)
   - total_sync_runs (count distinct time windows in audit log)
   - total_successful_actions (action entries where success=true and action != skipped_locked)
   - total_failed_actions (action entries where success=false or action_taken=failed)
   - total_skipped_locked (action entries where action_taken=skipped_locked)
   - total_retries_from_error_log (count from sync_errors file for today)
   - unresolved_failures (from failed_students.json if date matches today)
5. Write summary to daily_summary_{YYYY-MM-DD}.txt and .json

**Part B: Call generate_daily_summary at end of day in run_attendance_sync.py**

In sync_attendance_to_aeries(), after the verification report step, add:

```python
# Step 4: Generate daily summary (only at END OF DAY sync)
if "END OF DAY" in sync_label:
    logger.info("")
    logger.info("STEP 4: Generating Daily Summary")
    logger.info("-" * 70)
    try:
        from sync_utils import generate_daily_summary
        summary = generate_daily_summary(today, output_dir=".")
        logger.info(f"   Total sync runs today: {summary['total_sync_runs']}")
        logger.info(f"   Total students processed: {summary['total_students_processed']}")
        logger.info(f"   Successful: {summary['total_successful_actions']}")
        logger.info(f"   Failed: {summary['total_failed_actions']}")
        logger.info(f"   Locked (skipped): {summary['total_skipped_locked']}")
        if summary.get('unresolved_failures'):
            logger.warning(f"   !! UNRESOLVED: {len(summary['unresolved_failures'])} students")
    except Exception as e:
        logger.error(f"   Failed to generate daily summary: {e}")
```

Import generate_daily_summary at the top of run_attendance_sync.py alongside other sync_utils imports.
